PROJECT 2 : SMART TROLLEY AUTOMATION

Developed an IoT-powered smart shopping cart system to streamline retail checkout processes using RFID technology. Integrated ESP8266 with RFID modules to detect product details in real time as customers add or remove items. Built a Django-based backend to manage inventory, user accounts, and transaction logs, enabling instant billing and reducing human intervention. The system improves accuracy, reduces wait times, and offers a smoother shopping experience through automated data handling and a customer-friendly interface. Technologies Used: Python, Django, MySQL, HTML, CSS, JS, IOT module(ESP8266, RFID Reader, RFID Tag).
